5. 🖥️ FRONTEND - SCHERMI

5.1 Cucina (cucina.html) - TOUCH SCREEN

URL: http://IP:8080/cucina.html

Funzionalità principali:

  • Griglia 60 tavoli (6 colonne x 10 righe)
  • Selezione multipla (tocca i tavoli per selezionarli/deselezionarli)
  • Barra azioni in basso (appare solo con tavoli selezionati)
  • Statistiche in tempo reale (In attesa/In lavoro/Pronti/Non disp.)
  • Aggiornamento automatico ogni 4 secondi
  • Alert vocale (feedback sonoro)

Pulsanti azione:

PulsanteColoreAzioneVocale
PRONTO🟢 VerdeStato → pronto"Tavolo X, pronto da portare"
IN LAVORAZIONE🟠 ArancioneMostra scelta tempo (10/20/30/45/60 min)"Tavolo X, in lavorazione per Y minuti"
NON DISPONIBILE🔴 RossoStato → non disponibileNessuna
ANNULLA⚪ GrigioDeseleziona tuttiNessuna

5.2 Cassa (cassa.html) - MONITOR

URL: http://IP:8080/cassa.html

  • Griglia 60 tavoli in sola lettura
  • Mostra per ogni tavolo: numero, stato, tempo (se presente)
  • Aggiornamento automatico ogni 5 secondi
  • Nessun pulsante di azione (solo visualizzazione)

5.3 Storico (storico.html)

URL: http://IP:8080/storico.html

  • Tabella con tutti i cambi di stato
  • Colonne: Tavolo, Stato, Tempo, Data/Ora
  • Filtro per tavolo (select menu)
  • Ordinamento cronologico decrescente (il più recente in alto)

11. 🔧 PERSONALIZZAZIONI COMUNI

11.1 Cambiare il numero di tavoli (es. da 60 a 80)

for i in $(seq 61 80); do
  docker exec -i pizzeria_mysql mysql -upizzeria_user -ppizza123 -e \
  "USE pizzeria; INSERT INTO tavoli (id, nome) VALUES ($i, 'Tavolo $i');"
  docker exec -i pizzeria_mysql mysql -upizzeria_user -ppizza123 -e \
  "USE pizzeria; INSERT INTO stati_tavoli (tavolo_id, stato) VALUES ($i, 'in_attesa');"
done

11.2 Cambiare il testo dell'alert vocale

Modificare in frontend/cucina.html le righe:

// Prima
parla(`Tavolo ${id}, pronto da portare`);

// Dopo - esempio
parla(`Il tavolo ${id} è pronto, venite a ritirare`);

11.3 Cambiare l'IP del server

Modificare in TUTTI i file .html la riga:

let API_URL = 'http://' + window.location.hostname + ':3000';

Sostituire con IP fisso:

let API_URL = 'http://192.168.1.100:3000';

11.4 Cambiare i colori degli stati

Modificare in frontend/cucina.html i CSS:

.stato-pronto { background: #28a745; }  /* Verde */
.stato-lavorazione { background: #fd7e14; }  /* Arancione */
.stato-non-disponibile { background: #dc3545; }  /* Rosso */
.stato-attesa { background: #6c757d; }  /* Grigio */

14. 🔌 API ESEMPI PRATICI

Ottenere stato di tutti i tavoli

curl http://localhost:3000/api/stati-tavoli

Impostare tavolo 5 come "pronto" (con storico)

curl -X PUT http://localhost:3000/api/stati-tavoli/5/con-storico \
  -H "Content-Type: application/json" \
  -d '{"stato":"pronto","tempo_minuti":null}'

Impostare tavolo 5 come "in lavorazione" per 20 minuti

curl -X PUT http://localhost:3000/api/stati-tavoli/5/con-storico \
  -H "Content-Type: application/json" \
  -d '{"stato":"lavorazione","tempo_minuti":20}'

Ottenere tutto lo storico

curl http://localhost:3000/api/storico-tutti
